

# Updated Plan — Name Change to WebStorey

The only change from the approved plan is the brand name. Everything else remains identical.

## What changes

- **Company name**: PixelForge → **WebStorey**
- **Logo**: Geometric "W" monogram with the same vibrant gradient (purple → blue → cyan)
- All references throughout the site (nav, hero, footer, meta tags, page title) will use **WebStorey**
- Tagline adjusted to fit the new name (e.g., "We Build Your Web Story")

## Everything else stays the same

All sections (Hero, Services, Stats, Portfolio, Testimonials, Pricing, Contact, Footer), the bold & colorful design, i18n with full RTL Hebrew support, accessibility panel, scroll animations, parallax effects, and all other approved features remain unchanged.
